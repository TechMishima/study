# TIL
## プロパティ
### widthプロパティ
内容領域の幅を指定
```
.text {
  width: 80%;
}
```

### heightプロパティ
内容領域の高さを指定
```
.text {
  height: 300px;
}
```

### max-widthプロパティ max-heightプロパティ
内容領域の最大を指定
```
.text {
  max-width: 500px;
  max-height: 500px;
}
```

### max-widthプロパティ max-heightプロパティ
内容領域の最小を指定
```
.text {
  mix-width: 500px;
  mix-height: 500px;
}
```
