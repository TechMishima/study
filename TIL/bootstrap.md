### html5
BootstrapはHTML5doctypeを使用する必要がある。<br>
使用しないと,不完全にスタイリングされ大きな問題を引き起こす可能性がある。
```
<!doctype html>
<html lang="ja">
  ...
</html>
```

### レスポンシブ用メタタグ
Bootstrapはモバイル端末で使いやすいようにコードが最適化されている。
```
<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
```

### もしもボックスサイズ指定ができない場合は？
デフォルトではボックスサイズがborder-boxになっている
もし変更したい時は下記を使用する。
```
.selector-for-some-widget {
  box-sizing: content-box;
}
```
