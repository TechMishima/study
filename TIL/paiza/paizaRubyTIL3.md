# Til
### 小数点以下の扱い
四捨五入 round
```
100.3333.round # 出力結果 ==> 100
100.3333.round(1) # 出力結果 ==> 100.3
```
小数点以下切り上げ ceil
```
100.3333.ceil # 出力結果 ==> 101
100.3333.ceil(1) # 出力結果 ==> 100.4
```
小数点以下切り捨て floor
```
100.3333.floor # 出力結果 ==> 100
100.3333.floor(1) # 出力結果 ==> 100.3
```
