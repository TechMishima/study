### 戻り値はなに？

```
問題
public static Notifier getInstance() { return なんぞ; }
```

1. Notifier型に入るのは何か考える

2. クラス内にあるNotifier型の宣言をみる。

```
private static final Notifier INSTANCE = new Notifier();
```
変数はINSTANCEしかない。

> 答えは INSTANCE

★★秒殺

```
public static ○○ getInstance()
これはINSTANCEが答え
```

### ★変数宣言がされているかチェック for-each

```
for (OrderBy orderBy : orderBys) {

}
```
上記では後に変数宣言されていないOrderByがあるため、forで宣言しなくてはならない

### ★なんとか?

```
orderBy.isReversed ? -order : order
```
上記はorderByの真偽で判定が変わる。
- orderByのisReversedがtrueであれば、「左」の-orderになる。
- orderByのisReversedがfalseであれば、「右」のorderになる。

※★メソッドだと()がつくから判断ができる！！！
